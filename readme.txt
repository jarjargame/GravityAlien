■このゲームについて
落ちものパズルゲームです。エイリアンを消してハイスコアを目指しましょう。

■起動方法
ソースコードからゲームを実行するには「pygame」をインストールする必要があります。

■基本操作(デフォルト)
ゲームを始めるには起動後のタイトル画面で[p]キーを押してください。
詳しくは同梱のhowtoplay.txtを参照するか、タイトル画面で[h]キーを押してください。


■謝辞
credit.txtを参照するか、タイトル画面で[c]キーを押してください。
